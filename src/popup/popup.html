Canvas - detect and download.
Settings

Requires "Ask where to save each file" to be OFF in Chrome settings. Will download the file automatically to the default location (Downloads folder).

Download each file marked with a checkbox separately instead of creating a ZIP archive. Works best with "Use default download location" enabled since it allows automatic saving of large amounts of files without prompts.

Use timestamp (YYYYMMDD_HHMMSS) instead of "canvas" prefix for file names. Provides better sorting capabilities and helps organize files created chronologically.

Try to extract images from data-src, data-source, data-url attributes on tainted canvases. This can help recover images from cross-origin canvases when the original URL is available.
